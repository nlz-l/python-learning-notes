# 南方电网电力负荷预测项目

本项目基于历史电力负荷及相关特征数据，训练 XGBoost 模型，实现多变量单步电力负荷预测。项目保留了数据、训练日志、模型文件和核心业务代码，适合学习时间序列特征工程和机器学习建模流程。

## 项目结构

```text
data/     原始数据、处理后的训练数据或预测输入数据
log/      训练、评估或运行过程中产生的日志
model/    保存训练完成的模型文件
src/      项目核心代码，包括数据处理、模型训练、预测等逻辑
utils/    自定义工具函数，例如日志、配置、指标或通用处理函数
```

## 建模思路

1. 读取历史负荷数据和相关影响因素。
2. 完成缺失值处理、特征选择、时间特征构造等数据预处理。
3. 使用 XGBoost 训练回归预测模型。
4. 保存模型文件，便于后续复用或部署。
5. 使用测试集或新数据进行单步预测，并评估预测误差。

## 推荐环境

- Python 3.10 或更高版本
- 常用依赖：

```bash
pip install pandas numpy scikit-learn xgboost matplotlib joblib
```

实际依赖请以 `src/` 中的导入语句为准。

## 运行建议

由于该项目按建模流程拆分代码，建议先阅读 `src/` 目录，确认入口脚本名称和数据路径后再运行。例如：

```bash
python src/训练入口脚本.py
```

如果运行时提示找不到数据文件，请检查当前工作目录是否为项目根目录，或在代码中改为绝对路径。

## 学习重点

- 电力负荷预测问题的特征构造方式
- XGBoost 在回归任务中的训练与调参
- 模型文件保存、加载和复用
- 日志记录和实验结果管理
- 从课程案例整理为可复用项目结构的方法
