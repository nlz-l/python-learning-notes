# 1 绪论

## 1.1 研究背景

从互联网普及和大学生数字化生活方式切入，说明网络已成为大学生学习、社交、娱乐和生活管理的重要工具。  
同时指出过度网络使用可能带来的负面影响，例如注意力分散、学习效率下降、作业拖延、睡眠不足、焦虑水平升高、线下社交减少等。

可写内容：

- 互联网和智能终端在大学生群体中的普及现状。
- 网络使用对大学生学习资源获取、信息检索、在线课程参与的积极作用。
- 过度使用网络对学习状态和身心健康的潜在影响。
- 高校加强网络使用引导和健康教育的现实需求。

## 1.2 研究目的及意义

### 1.2.1 研究目的

本研究旨在通过多源数据分析，系统探究大学生网络使用深度与学习状态、身心健康之间的关系，并进一步分析不同群体之间是否存在差异，以及学习状态是否在网络使用与身心健康之间发挥中介作用。

主要研究目的包括：

1. 描述大学生网络使用行为的基本特征。
2. 分析网络使用深度与学习状态之间的关系。
3. 分析网络使用深度与身心健康之间的关系。
4. 探究学习状态与身心健康之间的关联。
5. 比较不同性别、学历、年级等群体之间的差异。
6. 构建回归模型和中介效应模型，解释变量之间的作用路径。
7. 为高校开展网络行为引导和学生健康管理提供数据依据。

### 1.2.2 研究意义

#### 理论意义

- 丰富大学生网络行为、学习状态和身心健康关系的实证研究。
- 验证网络使用深度、学习状态和身心健康之间可能存在的关联机制。
- 探索多源数据在学生行为研究中的应用价值。

#### 实践意义

- 为高校制定科学的网络使用管理和引导策略提供依据。
- 帮助大学生认识网络过度使用对学习和健康的潜在影响。
- 为心理健康教育、学业指导和生活管理提供参考。

## 1.3 国内外研究现状

本部分可围绕以下方向展开综述：

### 1.3.1 大学生网络使用行为研究

概述国内外关于大学生网络使用时长、网络依赖、短视频使用、社交媒体使用、游戏行为等方面的研究。

### 1.3.2 网络使用与学习状态关系研究

重点综述网络使用对课堂注意力、课外学习时长、学习效率、作业完成质量、学业成绩等方面的影响。

重点综述网络使用与睡眠质量、焦虑、压力、抑郁、身体不适、体育锻炼和线下社交之间的关系。

### 1.3.4 多源数据在学生行为研究中的应用

说明问卷数据、公开数据和网络爬虫数据各自的优势与局限，突出多源数据交叉验证的必要性。

# 2 研究内容与调查方案

## 2.1 研究对象

本研究对象为在校大学生群体，涵盖不同性别、年级、专业类别、是否住校、家庭所在地、学习成绩自评等特征，以保证样本具有一定的多样性和代表性。

可写内容：

- 研究对象的范围：在校大学生。
- 样本覆盖的基本特征：性别、年级、专业类别、住宿情况、家庭所在地等。
- 样本选择的原因：大学生网络使用频繁，学习和生活均与网络高度相关。

## 2.2 研究内容

本研究主要围绕以下四个模块展开：

### 2.2.1 基本信息模块

包括性别、年级、专业类别、是否住校、家庭所在地、家庭收入水平、学习成绩自评、兼职情况、恋爱关系等变量。

### 2.2.2 网络使用情况模块

包括每日平均上网时长、学习日上网时长、周末上网时长、睡前手机使用天数、游戏时长、短视频时长、社交媒体时长、影视直播时长、学习类上网时长、网络依赖程度、断网焦虑程度等变量。

### 2.2.3 学习状态模块

包括课堂注意力、课堂手机使用频率、上课走神频率、课外学习时长、作业完成质量、学习时分心频率、娱乐后学习困难程度、学习效率下降程度、熬夜赶作业频率、学习状态满意度等变量。

### 2.2.4 身心健康与生活状态模块

包括自评健康状况、眼睛不适、肩颈不适、头痛头晕、睡眠时长、入睡时间、熬夜天数、睡眠质量、焦虑程度、压力程度、抑郁程度、线下社交频率、人际关系满意度、体育锻炼频率、饮食规律程度等变量。

## 2.3 研究问题

本研究设置以下五个核心研究问题：

- RQ1：大学生网络使用深度是否对学习状态产生负面影响？
- RQ2：大学生网络使用深度是否对身心健康产生负面影响？
- RQ3：学习状态与身心健康之间是否存在关联？
- RQ4：网络使用深度、学习状态、身心健康在不同人口学群体间是否存在差异？
- RQ5：学习状态是否在网络使用深度与身心健康之间起中介作用？

## 2.4 研究假设

基于研究问题，提出以下假设：

- H1：网络使用深度对学习状态有显著负向影响。
- H2：网络使用深度对身心健康有显著负向影响。
- H3：学习状态与身心健康呈正相关。
- H4：网络使用深度、学习状态、身心健康在性别、学历等群体间存在显著差异。
- H5：学习状态在网络使用深度与身心健康之间起中介作用。

## 2.5 调查方法

本研究采用三源数据采集策略。

### 2.5.1 问卷法

通过问卷星平台发放线上问卷，并辅以线下纸质问卷，收集大学生关于网络使用、学习状态和身心健康的自评数据。

### 2.5.2 网络爬虫法

使用 Python 爬虫程序从公开在线平台采集大学生网络使用行为及相关信息，形成网络爬虫数据集。

### 2.5.3 二手数据分析法

选取 Kaggle 开源数据集作为二手数据来源，用于与一手问卷和爬虫数据进行对照分析。

## 2.6 抽样方法与实施过程

可说明：

1. 问卷通过社交平台和线下场所发放。
2. 设置 IP 限制或设备限制，减少重复填答。
3. 所有题目设置为必答题，降低缺失值风险。
4. 爬虫数据仅采集公开平台数据。
5. 二手数据来源公开，便于复现和验证。

# 3 数据来源与整理

## 3.1 数据来源概况

本研究共使用三份数据源：

| 数据源 | 来源 | 样本量 | 主要变量 | 作用 |
|---|---:|---:|---|---|
| 一手问卷数据 | 问卷星与线下问卷 | 400 | 网络使用、学习状态、身心健康、人口学变量 | 探索性分析 |
| 二手公开数据 | Kaggle 开源数据集 | 1200 | 屏幕时长、社交媒体使用、睡眠、运动、压力、成绩 | 验证性分析 |
| 网络爬虫数据 | Python 爬虫采集 | 705 | 网络使用、学习参与、健康自述 | 主建模数据 |

## 3.2 一手问卷数据整理

一手问卷数据共回收有效样本 400 份。原始问卷约包含 60 个题项，涵盖基本信息、网络使用情况、学习状态和身心健康四个模块。

整理过程包括：

1. 原始列名重命名。
2. 分类变量编码。
3. 反向题进行正向转换。
4. 检查缺失值、重复值和异常值。
5. 合成网络使用深度得分、学习状态得分和身心健康得分。
6. 输出清洗后的数据文件。

## 3.3 二手公开数据整理

二手数据来自 Kaggle 开源数据集，样本量为 1200。原始变量包括年龄、性别、日均屏幕时长、社交媒体时长、AI 工具使用时长、睡眠时长、运动时长、心理压力等级和学业成绩等。

整理过程包括：

1. 英文变量名转换为中文变量名。
2. 性别等分类变量重新编码。
3. 心理压力等级进行反向计分。
4. 对相关变量进行标准化。
5. 合成三个统一维度得分。
6. 输出清洗后的数据文件。

## 3.4 网络爬虫数据整理

网络爬虫数据通过 Python 程序从公开在线平台采集，清洗后得到 705 条有效记录。

整理过程包括：

1. 删除缺失严重或无效记录。
2. 对分类变量进行编码。
3. 提取网络使用、学习参与、健康自述等结构化信息。
4. 对不同量纲变量进行标准化处理。
5. 合成网络使用深度、学习状态和身心健康三个维度得分。
6. 输出清洗后的数据文件。

## 3.5 数据预处理流程

三份数据均经过以下预处理步骤：

1. 列名规范化。
2. 缺失值检查。
3. 重复值检查。
4. 异常值筛查。
5. 分类变量数值编码。
6. 反向题计分转换。
7. 维度得分合成。
8. 标准化处理。
9. 输出清洗数据集。

## 3.6 数据质量评估

从以下方面进行质量评估：

### 3.6.1 缺失值检查

说明三份数据均不存在严重缺失，具备后续分析基础。

### 3.6.2 异常值检测

采用 IQR 四分位距法进行异常值检测，对轻微异常值进行保留，对极端异常值进行进一步检查。

### 3.6.3 正态性检验

采用偏度、峰度、Shapiro-Wilk 检验、D'Agostino K² 检验和可视化图形判断变量分布。

### 3.6.4 共同方法偏差检验

针对一手问卷数据，采用 Harman 单因子检验判断是否存在严重共同方法偏差。

### 3.6.5 信度分析

对问卷中各维度进行 Cronbach's α 检验。若系数偏低，可解释为部分维度属于形成性综合指数，内部题项反映不同侧面，不能简单按反映性量表理解。

# 4 建模前检验

## 4.1 正态性检验

对三份数据中的网络使用深度得分、学习状态得分和身心健康得分进行正态性检验。

可写内容：

- 使用 Shapiro-Wilk 检验和 Q-Q 图进行判断。
- 三份数据的部分维度得分不服从正态分布。
- 后续分析需根据数据特征选择参数检验、非参数检验或 Bootstrap 方法。

## 4.2 方差齐性检验

采用 Levene 检验，按性别、年级、学历、是否住校、成绩自评等变量进行方差齐性检验。

可写内容：

- 一手问卷和二手数据大部分分组满足方差齐性。
- 爬虫数据在部分性别和学历分组下存在方差不齐。
- 后续分组比较中可采用 Welch 校正或 Kruskal-Wallis 非参数检验作为补充。

## 4.3 相关分析

对三个核心维度进行 Pearson 和 Spearman 相关分析。

### 4.3.1 一手问卷数据相关分析

一手问卷中网络使用深度、学习状态和身心健康三者几乎零相关，说明三维度间未呈现明显线性关系。

### 4.3.2 二手数据相关分析

二手数据中三个维度也呈现较弱相关，说明其不适合用于传统回归建模。

### 4.3.3 爬虫数据相关分析

爬虫数据中三维度关系符合理论预期：

- 网络使用深度与学习状态呈显著负相关。
- 网络使用深度与身心健康呈显著负相关。
- 学习状态与身心健康呈显著正相关。

这一发现说明爬虫数据适合作为后续多元回归和中介效应分析的主建模数据。

## 4.4 共线性诊断

对爬虫数据中的关键自变量进行 VIF 检验。  
若 VIF 小于 5，则说明不存在严重多重共线性，可进入多元线性回归和中介模型。

## 4.5 分组差异检验

采用独立样本 t 检验和单因素方差分析，比较不同人口学群体之间的差异。

分析对象包括：

1. 性别差异。
2. 年级差异。
3. 学历差异。
4. 是否住校差异。
5. 成绩自评差异。

重点说明：

- 一手问卷和二手数据中的分组差异不明显。
- 爬虫数据中性别和学历差异较为显著。
- 性别和学历可在后续模型中作为控制变量或调节变量。

## 4.6 前置检验小结

本部分应总结建模前检验的核心结论：

1. 三份数据整体质量可接受。
2. 数据分布并非完全正态，需要稳健方法补充。
3. 一手问卷和二手数据中三个维度间几乎无线性相关。
4. 爬虫数据中变量关系显著，适合作为主建模数据。
5. 性别、学历等变量可作为控制变量或调节变量。
6. 后续建模应采用分数据源差异化策略。

### 5.1.1 爬虫数据：主建模数据

爬虫数据中变量关系显著，适合用于：

1. 描述性统计。
2. 相关分析。
3. 分组差异检验。
4. 多元线性回归。
5. 中介效应分析。
6. 调节效应分析。
7. 模型诊断与效应量报告。

### 5.1.2 一手问卷数据：探索性分析

一手问卷三维度间几乎零相关，不适合强行回归。可进行：

1. 题项级相关分析。
2. 维度内部结构分析。
3. 网络使用行为聚类。
4. 不同聚类群体的学习与健康差异比较。

二手数据变量较少，主要用于：

1. 大样本描述性统计。
2. 性别分组差异检验。
3. 与一手问卷和爬虫数据进行横向对比。
4. 验证研究结论的稳健性。

## 5.3 多元线性回归模型

以爬虫数据为主，构建三个回归模型。

### 5.3.1 模型一：网络使用深度对学习状态的影响

因变量：学习状态得分。  
自变量：网络使用深度。  
控制变量：性别、学历。  

模型表达式：

```text
学习状态得分 = β0 + β1 × 网络使用深度 + β2 × 性别 + β3 × 学历 + ε
```

分析重点：

- 网络使用深度的回归系数是否显著。
- 系数方向是否为负。
- 模型解释力 R² 和调整 R²。
- 控制变量是否显著。

### 5.3.2 模型二：网络使用深度对身心健康的影响

因变量：身心健康得分。  
自变量：网络使用深度。  
控制变量：性别、学历。  

模型表达式：

```text
身心健康得分 = β0 + β1 × 网络使用深度 + β2 × 性别 + β3 × 学历 + ε
```

分析重点：

- 网络使用深度是否显著负向预测身心健康。
- 性别、学历等控制变量是否影响结果。
- 模型整体显著性。

### 5.3.3 模型三：加入学习状态后的身心健康模型

因变量：身心健康得分。  
自变量：网络使用深度、学习状态得分。  
控制变量：性别、学历。  

模型表达式：

```text
身心健康得分 = β0 + β1 × 网络使用深度 + β2 × 学习状态得分 + β3 × 性别 + β4 × 学历 + ε
```

分析重点：

- 学习状态对身心健康是否具有显著预测作用。
- 加入学习状态后，网络使用深度的系数是否减弱。
- 模型三相较模型二的 R² 是否提高。
- 是否为中介效应分析提供支持。

## 5.4 中介效应分析

构建中介路径：

```text
网络使用深度 → 学习状态 → 身心健康
```

### 5.4.1 Baron & Kenny 三步法

#### 第一步：总效应检验

检验网络使用深度对身心健康的总效应。

```text
身心健康 = c0 + c × 网络使用深度 + e1
```

#### 第二步：自变量对中介变量的影响

检验网络使用深度对学习状态的影响。

```text
学习状态 = a0 + a × 网络使用深度 + e2
```

#### 第三步：加入中介变量后的直接效应

检验网络使用深度和学习状态共同对身心健康的影响。

```text
身心健康 = c'0 + c' × 网络使用深度 + b × 学习状态 + e3
```

### 5.4.2 Bootstrap 置信区间法

采用 5000 次 Bootstrap 重抽样，检验间接效应 `a × b` 的 95% 置信区间是否包含 0。

判断标准：

- 若置信区间不包含 0，说明中介效应显著。
- 若加入学习状态后，网络使用深度不再显著，说明可能存在完全中介。
- 若网络使用深度仍显著但系数减小，说明可能存在部分中介。

## 5.5 调节效应分析

以性别和学历作为候选调节变量，检验网络使用深度对学习状态或身心健康的影响是否因群体不同而变化。

### 5.5.1 性别调节模型

```text
学习状态 = β0 + β1 × 网络使用深度 + β2 × 性别 + β3 × 网络使用深度 × 性别 + ε
```

若交互项显著，说明性别对网络使用与学习状态之间的关系具有调节作用。

### 5.5.2 学历调节模型

```text
学习状态 = β0 + β1 × 网络使用深度 + β2 × 学历 + β3 × 网络使用深度 × 学历 + ε
```

若交互项显著，说明学历对网络使用影响学习状态的关系具有调节作用。

### 5.5.3 分组回归

可按不同学历或性别分组，分别建立回归模型，比较不同群体中回归系数的方向和强度。

## 5.6 一手问卷探索性分析

由于一手问卷三维度间缺乏显著线性关系，本部分不进行强行回归，而转向题项级和行为类型分析。

### 5.6.1 题项级相关分析

分析具体网络行为与具体学习、健康题项之间的关系，例如：

- 游戏时长与学习分心频率。
- 短视频时长与睡眠质量。
- 社交媒体时长与焦虑程度。
- 睡前手机使用与入睡困难。
- 网络依赖程度与学习状态满意度。

### 5.6.2 维度内部结构分析

分析每个维度内部不同题项之间的关系，判断维度内部是否存在多个子结构。

### 5.6.3 聚类分析

基于网络使用行为对学生进行聚类，例如：

1. 低使用型。
2. 学习导向型。
3. 娱乐偏好型。
4. 高依赖型。

再比较不同类型学生在学习状态和身心健康方面的差异。

### 5.6.4 探索性分析小结

总结问卷数据中的具体行为模式，并解释为什么维度合成后可能未呈现显著相关。

## 5.7 二手数据验证性分析

二手数据主要用于对研究发现进行补充验证。

分析内容包括：

1. 样本基本特征统计。
2. 性别分组差异检验。
3. 网络使用变量与睡眠、运动、压力、成绩的简单相关。
4. 与爬虫数据和一手问卷的横向比较。
5. 分析二手数据未出现显著相关的可能原因。

## 5.8 模型诊断与评估

对回归模型进行系统诊断。

| 诊断项目 | 方法 | 判断标准 |

# 6 结果讨论

## 6.1 主要研究发现

本部分总结研究结果，可从以下方面展开：

1. 三份数据均能从不同角度反映大学生网络使用、学习状态和身心健康情况。
2. 爬虫数据中网络使用深度与学习状态、身心健康呈显著负相关。
3. 爬虫数据中学习状态与身心健康呈显著正相关。
4. 一手问卷和二手数据中三维度间几乎零相关。
5. 性别、学历等变量在爬虫数据中可能存在显著分组差异。
6. 学习状态可能在网络使用深度与身心健康之间发挥中介作用。

## 6.2 多源数据结果差异分析

可从以下角度解释三份数据结果不一致的原因：

### 6.2.1 数据来源差异

一手问卷依赖学生自我报告，可能受到主观判断、社会期许和填答习惯影响；爬虫数据更接近公开行为表现；二手数据变量较少，难以完整刻画复杂关系。

### 6.2.2 变量测量差异

不同数据源对“网络使用”“学习状态”和“身心健康”的测量方式不同，导致维度合成结果和变量关系存在差异。

### 6.2.3 样本结构差异

三份数据的样本来源、样本范围和群体结构不同，可能影响结果的可比性。

### 6.2.4 维度合成方式差异

一手问卷题项较多，维度内部可能包含多个子方面；如果直接合成为一个总分，可能掩盖具体题项之间的真实关系。

## 6.3 与已有研究的比较

将本研究结果与已有文献进行比较：

1. 是否支持过度网络使用会降低学习效率的观点。
2. 是否支持网络使用与睡眠、焦虑、压力有关的观点。
3. 是否支持学习状态与心理健康相互关联的观点。
4. 本研究中多源数据结果不一致对已有研究的补充价值。

## 6.4 理论启示

理论上，本研究说明：

1. 网络使用对大学生学习与健康的影响并非简单线性关系。
2. 学习状态可能是网络使用影响身心健康的重要中介变量。
3. 多源数据可以揭示单一问卷数据难以发现的关系模式。
4. 形成性综合指数与反映性量表在解释时应加以区分。

## 6.5 实践启示

针对高校和学生提出建议：

1. 合理引导学生控制娱乐性网络使用。
2. 关注短视频、游戏、社交媒体等高沉浸网络活动。
3. 加强学生时间管理和注意力管理教育。
4. 建立睡眠健康和心理健康监测机制。
5. 鼓励学生增加线下社交和体育锻炼。
6. 对高网络依赖学生开展针对性干预。

## 6.6 研究局限

本研究仍存在以下局限：

1. 横截面数据不能严格推断因果关系。
2. 一手问卷数据可能存在主观填答偏差。
3. 二手数据变量较少，解释力有限。
4. 爬虫数据来源于公开平台，样本代表性仍需谨慎判断。
5. 维度合成方式可能影响变量关系。
6. 未能充分控制所有潜在混淆变量。

## 6.7 后续研究展望

未来研究可以从以下方向拓展：

1. 采用纵向追踪数据验证因果关系。
2. 优化问卷量表结构，提高信度和效度。
3. 增加更多客观行为数据，如手机使用记录、学习平台日志等。
4. 结合访谈法深入理解学生网络使用动机。
5. 构建更复杂的结构方程模型。
6. 针对不同专业、年级和性别群体进行细分研究。

# 7 结论

本研究基于一手问卷数据、二手公开数据和网络爬虫数据，系统分析了大学生网络使用对学习状态与身心健康的影响。

可总结为以下几点：

1. 大学生网络使用已深度融入学习与生活，对学习状态和身心健康具有重要影响。
2. 三份数据源在变量关系上呈现明显差异，其中爬虫数据最符合理论预期。
3. 爬虫数据中，网络使用深度与学习状态、身心健康呈显著负相关。
4. 学习状态与身心健康呈显著正相关。
5. 一手问卷和二手数据中三维度间相关较弱，不适合强行进行回归建模。
6. 学习状态可能在网络使用深度与身心健康之间发挥中介作用。
7. 性别、学历等人口学变量可能影响网络使用及其结果。
8. 本研究结果提示高校应加强大学生网络使用引导、学习管理和心理健康支持。
9. 由于数据为横截面数据，研究结论应表述为变量之间的关联，而非严格因果关系。
