# 数据建模计划
## 大学生网络使用对学习状态与身心健康的影响

基于前置检验结果，制定分数据源的建模策略与分析方案。

---

## 1. 研究问题与假设

### 核心研究问题

- **RQ1**：大学生网络使用深度是否对学习状态产生负面影响？
- **RQ2**：大学生网络使用深度是否对身心健康产生负面影响？
- **RQ3**：学习状态与身心健康之间是否存在关联？
- **RQ4**：网络使用深度、学习状态、身心健康在不同人口学群体间是否存在差异？
- **RQ5**：学习状态是否在网络使用深度与身心健康之间起中介作用？

### 理论假设

| 假设 | 内容 | 前置检验预判 |
|------|------|-------------|
| H1 | 网络使用深度对学习状态有显著负向影响 | 爬虫数据支持(r=-0.499\*\*\*)；一手/二手不支持(r≈0) |
| H2 | 网络使用深度对身心健康有显著负向影响 | 爬虫数据支持(r=-0.658\*\*\*)；一手/二手不支持(r≈0) |
| H3 | 学习状态与身心健康呈正相关 | 爬虫数据支持(r=0.782\*\*\*)；一手/二手不支持(r≈0) |
| H4 | 网络使用/学习/健康在性别、学历等群体间存在差异 | 爬虫数据中性别和学历差异显著；一手/二手不显著 |
| H5 | 学习状态在网络使用与身心健康间起中介作用 | 待爬虫数据中介分析验证 |

> **关键发现**：前置检验表明，一手问卷和二手数据的三维度间几乎零相关，这意味着 H1-H3 在这两份数据中**缺乏建模基础**。只有爬虫数据具备验证假说的条件。因此建模策略需**分数据源差异化处理**。

---

## 2. 拟采用的分析方法

### 2.1 描述性统计分析

**目的**：呈现三份数据的基本特征，为建模提供背景。

- 各维度得分的均值、标准差、分布形态（数据质量评估中已完成）
- 各人口学变量的频次分布
- 交叉表：性别 × 年级 × 维度得分分段
- 可视化：分组箱线图、小提琴图、均值 ± SE 误差棒图

### 2.2 相关分析（已完成）

前置检验已报告 Pearson 和 Spearman 相关系数。爬虫数据是唯一呈现中强相关的数据源，后续建模以爬虫数据为主。

### 2.3 分组差异检验（已完成）

前置检验已完成独立样本 t 检验（性别、住校）和单因素方差分析（年级、学历、成绩自评）。爬虫数据的学历差异（p < 0.01）和性别差异（网络使用 p < 0.01）将纳入后续模型作为控制变量或调节变量。

### 2.4 多元线性回归（爬虫数据）

**目的**：量化网络使用深度对学习状态/身心健康的独立影响，控制人口学混淆变量。

**模型 1 — 学习状态为因变量**：

$$\text{学习状态得分} = \beta_0 + \beta_1 \cdot \text{网络使用深度} + \beta_2 \cdot \text{性别} + \beta_3 \cdot \text{学历} + \varepsilon$$

**模型 2 — 身心健康为因变量**：

$$\text{身心健康得分} = \beta_0 + \beta_1 \cdot \text{网络使用深度} + \beta_2 \cdot \text{性别} + \beta_3 \cdot \text{学历} + \varepsilon$$

**模型 3 — 学习状态加入后**：

$$\text{身心健康得分} = \beta_0 + \beta_1 \cdot \text{网络使用深度} + \beta_2 \cdot \text{学习状态得分} + \beta_3 \cdot \text{性别} + \beta_4 \cdot \text{学历} + \varepsilon$$

**评估指标**：R²、调整 R²、F 检验、各系数的 t 检验及置信区间。模型 2→3 比较时关注 β₁ 的变化。

**前提条件检查**：

- 线性假设：散点图趋势判断
- 残差正态性：Q-Q 图
- 残差等方差：残差-拟合值散点图
- 无严重共线性：VIF（前置检验已确认 < 5）
- 残差独立性：Durbin-Watson 检验

### 2.5 中介效应分析（爬虫数据）

**目的**：检验学习状态是否在网络使用深度与身心健康之间起中介作用。

**理论路径**：网络使用深度 → 学习状态 → 身心健康

**方法**：Baron & Kenny (1986) 三步法 + Bootstrap 置信区间法

**三步回归**：

- **第 1 步（c 路径）**：网络使用深度 → 身心健康（总效应）

  $$\text{健康} = c_0 + c \cdot \text{网络} + e_1$$

- **第 2 步（a 路径）**：网络使用深度 → 学习状态

  $$\text{学习} = a_0 + a \cdot \text{网络} + e_2$$

- **第 3 步（c' 和 b 路径）**：网络使用深度 + 学习状态 → 身心健康

  $$\text{健康} = c'_0 + c' \cdot \text{网络} + b \cdot \text{学习} + e_3$$

**判断标准**：

- 若 a 显著且 b 显著，中介效应存在
- 若 c' 不显著，为完全中介；若 c' 仍显著但减小，为部分中介
- Bootstrap 法（5000 次重抽样）检验间接效应 ab 的 95% CI 是否不包含 0

**前置检验预判**：爬虫数据 r(网络,健康) = -0.658、r(网络,学习) = -0.499、r(学习,健康) = 0.782，满足 Baron-Kenny 三步法的启动条件。

### 2.6 调节效应分析（可选）

**目的**：检验性别、学历等人口学变量是否调节网络使用与学习/健康之间的关系。

**方法**：引入交互项

$$\text{学习状态} = \beta_0 + \beta_1 \cdot \text{网络} + \beta_2 \cdot \text{调节变量} + \beta_3 \cdot (\text{网络} \times \text{调节变量}) + \varepsilon$$

若交互项 β₃ 显著，说明分组变量对自变量-因变量关系有调节作用。前置检验中爬虫数据已发现学历分组差异显著，可作为候选调节变量。

当调节变量为分类变量时，采用**分组回归**作为补充：分别在 Undergraduate、Graduate、High School 子样本中拟合模型，比较系数方向和大小的变化。

## 3. 分数据源建模策略

### 3.1 爬虫数据（n = 705）—— 主建模数据

| 分析 | 方法 | 目的 |
|------|------|------|
| 描述性统计 | 均值/标准差/频次 | 刻画样本特征 |
| 相关分析 | Pearson + Spearman | 变量间关联 |
| 分组差异 | t 检验 / ANOVA | 人口学差异 |
| 多元回归 | OLS（3 个模型） | 检验 H1, H2, H3 |
| 中介效应 | Baron-Kenny + Bootstrap | 检验 H5 |
| 调节效应 | 交互项 / 分组回归 | 学历/性别的调节作用 |

选择理由：爬虫数据是唯一三维度间存在显著相关的数据源，具备回归和路径分析的前提条件。

### 3.2 一手问卷（n = 400）—— 探索性分析

| 分析 | 方法 | 目的 |
|------|------|------|
| 维度内部分析 | 子维度相关性、题项聚类 | 探究为何维度间零相关 |
| 分组差异 | t 检验 / ANOVA | 人口学差异（补充爬虫发现） |
| 题项级相关 | 个别题项的相关矩阵 | 寻找潜在的跨维度关联 |
| 聚类分析（可选） | KMeans / 层次聚类 | 基于题项的被试分类 |

选择理由：一手问卷维度间几乎零相关（|r| < 0.03），因此不进行回归预测。但其丰富的题项（60 题）和人口学变量可用于探索性分析——寻找网络使用的**哪些具体方面**（如游戏 vs 短视频 vs 社交媒体）与学习/健康的**哪些具体方面**存在关联。

### 3.3 二手数据（n = 1200）—— 验证性分析

| 分析 | 方法 | 目的 |
|------|------|------|
| 分组差异 | t 检验 / ANOVA | 性别差异 |
| 一致性检验 | 与爬虫数据结论对比 | 交叉验证 |

选择理由：二手数据变量较少（10 个原始变量），且维度间也几乎零相关（|r| < 0.02），回归建模缺乏前提。其主要价值在于大样本的分组差异检验和与爬虫结论的对照。

## 4. 模型评估与诊断

### 4.1 回归模型诊断清单

| 诊断项 | 方法 | 阈值 |
|--------|------|------|
| 线性假设 | 散点图 + 成分残差图 | 主观判断 |
| 残差正态性 | Q-Q 图 + Shapiro-Wilk 检验 | p > 0.05 |
| 残差等方差 | Breusch-Pagan 检验 + 残差-拟合值散点图 | p > 0.05 |
| 残差独立性 | Durbin-Watson 统计量 | 1.5 < DW < 2.5 |
| 异常值/强影响点 | Cook's D + 学生化残差 | Cook's D < 4/n |
| 共线性 | VIF | < 5 |

### 4.2 模型比较

- **嵌套模型比较**：模型 2 (仅网络) vs 模型 3 (网络+学习)，使用 F 检验和 ΔR² 判断学习状态是否显著提升模型解释力
- **信息准则**：AIC / BIC 辅助模型选择
- **交叉验证**：将爬虫数据的模型结果与一手/二手数据的同类分析进行定性比较

### 4.3 效应量报告

- 回归：标准化回归系数 β 和 Cohen f²（局部效应量）
- 分组比较：Cohen d（t 检验）/ η²（ANOVA）
- 中介效应：间接效应占总效应的比例 ab / c

---

## 5. 分析流程总结

### 执行顺序

**第一阶段 —— 爬虫数据（主分析）**

1. 描述性统计
2. 相关分析（前置检验已做）
3. 分组差异检验（前置检验已做）
4. 多元线性回归（模型 1-3）
5. 回归诊断（残差分析、DW、BP 检验）
6. 中介效应分析（Baron-Kenny + Bootstrap）
7. 调节效应分析（交互项 / 分组回归）
8. 效应量报告与结论

**第二阶段 —— 一手问卷（探索分析）**

1. 维度内题项间相关矩阵
2. 跨维度题项级相关（网络活动时长 × 学习/健康题项）
3. 聚类分析（基于网络使用行为模式分类被试）
4. 不同行为模式的群体在学习/健康上的差异
5. 与爬虫结论的定性对比

**第三阶段 —— 三份数据交叉验证**

1. 三份数据共有发现的汇总
2. 不一致发现的讨论
3. 方法学差异的解释（量表差异、样本差异、测量差异）

### 注意事项

1. **一手/二手数据不能用于回归预测**：前置检验已证明三维度间无线性关系，强行回归会得到不显著的 β 系数和极低的 R²，不应为"有结果"而做无效分析。如实报告这一发现本身就是有价值的学术贡献——它表明问卷和二手数据中的维度合成方式可能需要重新审视。

2. **爬虫数据的中介分析前提**：三步法的每一步都需独立满足线性回归前提，特别是第 3 步中网络使用深度和学习状态作为联合预测变量时，需检查两者的相关性是否引起共线性（VIF 已确认 < 5，安全）。

3. **因果推论的限制**：所有数据均为横截面数据，回归系数只能说明关联强度，不能严格推断因果关系。报告时需使用"关联""预测"而非"导致""引起"。
