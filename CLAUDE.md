# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指导。

## 仓库概述

这是一个Python学习仓库，包含教学材料、练习和竞赛题解。代码库分为四个主要部分：

1. **数据结构与python基础** - Python基础和数据结构
2. **数据分析** - 使用Jupyter notebook进行数据分析（NumPy、Pandas）
3. **机器学习** - 机器学习实验和练习
4. **蓝桥** - 蓝桥杯编程竞赛题目和解答

## 语言说明

- 所有文档、注释和笔记均使用中文
- 创建或修改内容时，请使用中文以保持一致性
- 变量名和代码可能使用英文或拼音

## 仓库结构

### 数据结构与python基础
包含渐进式Python学习材料：
- 1.面向对象基础 - 面向对象编程基础（类、继承、魔法方法）
- 2.面向对象高级 - 面向对象高级（封装、多态、学生管理系统）
- 3.闭包和装饰器 - 闭包和装饰器
- 4.网编和多线程 - 网络编程和多线程
- 5.生成器与正则表达式 - 生成器和正则表达式
- 6.数据结构与算法初步 - 数据结构（链表、二叉树）
- Python二级 - Python二级考试练习

### 数据分析
数据分析相关的Jupyter notebook：
- NumPy数组操作和运算
- Pandas DataFrame操作、数据读写
- 数据存储在 `data/` 子目录中

### 机器学习
包含大型数据集的机器学习实验：
- `data.txt` (52MB) - 大型训练数据集
- `house.csv` - 房价回归数据
- 机器学习实验的Jupyter notebook

### 蓝桥
蓝桥杯竞赛题目：
- `答案/` - 历届竞赛官方题解（第4-12届）
- `自练/` - 个人练习题解

## 使用此仓库

### 运行Python文件
```bash
python <filename>.py
```

### 运行Jupyter Notebook
```bash
jupyter notebook
# 或运行特定notebook：
jupyter notebook <notebook_name>.ipynb
```

### 文件组织规则
- 每个主题的文件按数字编号（01_、02_等）表示学习顺序
- 随堂笔记文件（Day01_随堂笔记.md等）包含课堂总结
- 示例文件以"example_"为前缀
