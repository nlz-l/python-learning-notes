# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指导。

## 仓库概述

这是一个Python学习仓库，包含教学材料、练习和竞赛题解。代码库分为六个主要部分：

1. **1.python基础与数据结构** - Python基础和数据结构
2. **2.数据分析** - 使用Jupyter notebook进行数据分析（NumPy、Pandas）
3. **3.机器学习** - 机器学习实验和练习
4. **4.爬虫** - Web爬虫练习和作业
5. **5.蓝桥** - 蓝桥杯编程竞赛题目和解答
6. **网页展示** - 个人简历等HTML页面

## 语言说明

- 所有文档、注释和笔记均使用中文
- 创建或修改内容时，请使用中文以保持一致性
- 变量名和代码可能使用英文或拼音
- **与用户的所有交流（包括回复、解释、提问）必须使用中文**

## 使用此仓库

### 运行Python文件
```bash
python <filename>.py
```

### 运行Jupyter Notebook
```bash
jupyter notebook
# 或运行特定notebook：
jupyter notebook <notebook_name>.ipynb
```

### 文件组织规则
- 每个主题的文件按数字编号（01_、02_等）表示学习顺序
- 蓝桥自练题目按届次存放，文件名用 `b1.py`-`b8.py` 或含题目名
- 示例文件以"example_"为前缀

## 开发注意事项

- 这是一个学习仓库，不是生产代码库
- 代码示例以教学为目的，可能刻意保持简单
- 修改示例时，保持清晰性优先于优化
- 添加新示例时保持顺序编号系统
- 蓝桥题解新题按届次放入对应 `自练/` 子目录
