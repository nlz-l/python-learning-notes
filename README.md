## 🧩 模块简介

### 1. Python 基础与数据结构

按学习顺序编号，每个模块附随堂笔记（`.md`）。

| 模块 | 主要内容 |
|------|----------|
| 面向对象基础 | 类的定义、`__init__` / `__str__` / `__del__` 魔法方法、单继承与多继承 |
| 面向对象高级 | 方法重写、封装（私有属性）、多态、抽象类、类方法与静态方法 |
| 闭包和装饰器 | 闭包原理、`nonlocal`、各类装饰器写法、深浅拷贝 |
| 网编和多线程 | TCP Socket 通信、文件上传、多进程、多线程、线程安全 |
| 生成器与正则 | 迭代器、`yield` 生成器、`property`、正则表达式匹配与提取 |
| 数据结构与算法 | 递归、链表、二叉树、冒泡/选择/插入/快速排序、二分查找、前缀和 |

### 3. 机器学习

**Boosting 集成学习对比实验**（`01_boosting集成学习对比实验.ipynb`）

对比 AdaBoost、GBDT、XGBoost、LightGBM 四种算法在乳腺癌数据集上的表现：

| 算法 | 准确率 | 训练耗时 |
|------|--------|----------|
| AdaBoost | 95.61% | 0.194s |
| GBDT | 95.61% | 0.328s |
| XGBoost | 94.74% | 0.174s |
| **LightGBM** | **97.37%** | **0.062s** |
