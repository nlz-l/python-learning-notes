### 3.1 模拟（Simulation）

**核心思想：** 按照题目描述的规则，一步一步模拟执行过程。

**适用场景：**

- 有明确的操作步骤和规则
- 不需要复杂的数据结构和优化
- 数据规模较小（通常 n ≤ 1000）

**模板：**

```python
def simulate():
    for round in range(total_rounds):   # 按回合循环
        for step in rules:             # 每回合内的操作
            execute(step)
            if condition:              # 判断是否满足条件
                update_state()
```

**十六届B1：攻击次数** — 模拟多回合战斗，每回合计算三种伤害 h₁(x), h₂(x)，扣减血量直到 ≤0，统计总回合数

**十六届B3：LQ图形** — 两段式循环打印：竖笔划 + 横笔划

***

### 3.2 贪心算法（Greedy）

**核心思想：** 每一步都选择当前最优解，最终得到全局最优解。不需要回溯。

**模板：**

```python
result = []
for item in sorted(items, key=rule):   # 按贪心策略排序
    if can_take(item):                   # 能选就选
        result.append(item)
```

**十六届B2：最长字符串** — 按长度从小到大排序，每个字符串只依赖前一个的前缀字符集合，能选就选，贪心取最长

**十六届B4：最多次数** — 从左到右扫描，遇到匹配的就"吃掉"，跳过已用的字符

***

### 3.3 枚举（Enumeration）

**核心思想：** 穷举所有可能的情况，逐一验证。

**十六届B6：枚举间距d** — 枚举所有可能的间距 d = 1, 2, ..., n-1，对每个d做一次LIS，取最大值

***

### 3.4 动态规划 DP（Dynamic Programming）

**核心思想：** 将大问题分解为重叠子问题，用记忆化避免重复计算。

**三要素：**

1. **状态定义：** dp\[i] 表示什么含义？
2. **状态转移方程：** dp\[i] 怎么从之前的状态推出来？
3. **初始条件和边界**

**十六届B6：LIS变种（等差递增子序列）**

- 状态定义：dp\[i] = 以第i个元素结尾的最长等差递增子序列长度
- 转移方程：`dp[i] = dp[i-d] + 1` （当 h\[i] > h\[i-d] 时）
- 关键区别于普通LIS：
  - 普通LIS：`dp[i] = max(dp[j]+1)` for `j < i` and `arr[j]<arr[i]`
  - 本题：`dp[i] = dp[i-d] + 1` （固定间距d，只需看前d个位置）

***

### 3.5 筛法（Sieve Method）

**核心思想：** 利用倍数关系批量更新，O(n log n) 预处理。

**十六届B5：约数个数筛法**

```python
dp = [0] * (L + 1)
for i in range(1, L + 1):        # i 从1开始！不是0！
    for j in range(i, L + 1, i): # j是i的倍数
        dp[j] += 1               # j多了一个约数i
```

追踪 L=6：

|  i  | 更新位置        | dp结果                         |
| :-: | ----------- | ---------------------------- |
|  1  | 1,2,3,4,5,6 | \[0,**1**,1,1,1,1,1]         |
|  2  | 2,4,6       | \[0,1,**2**,1,**2**,1,**2**] |
|  3  | 3,6         | \[0,1,2,**2**,2,1,**3**]     |
|  4  | 4           | \[0,1,2,2,**3**,1,3]         |
|  5  | 5           | \[0,1,2,2,3,**2**,3]         |
|  6  | 6           | \[0,1,2,2,3,2,**4**]         |

### 3.6 前缀和（Prefix Sum）

**核心思想：** 预处理前缀和数组，实现 O(1) 区间求和。

**一维前缀和：**

```python
s[i] = a[1] + a[2] + ... + a[i]     # 前i个数的和
s[i] = s[i-1] + a[i]                 # 递推公式

# 区间和：sum(a[l..r]) = s[r] - s[l-1]
```

**十六届B5应用：** dp\[] 存约数个数，s\[] 存前缀和。`s[k]` = 约数个数 ≤ k 的数有多少个？

***

### 3.7 容斥原理（Inclusion-Exclusion Principle）

**核心思想：** 求"同时满足多个条件"的数量时，用 总数 - 不满足A的 - 不满足B的 + 同时不满足AB的

**公式：** `|A ∩ B| = 总数 - |非A| - |非B| + |非A ∩ 非B|`

**韦恩图理解：**

```
        ┌─────────── 全部 ───────────┐
        │     ┌── 不含A ──┐          │
        │     │ ╲          │         │
        │     │   重叠区   │ 不含B   │
        │     └─────╲──────┘         │
        │          ╲ 有效区域        │
        └───────────────────────────┘
```

**十五届B2：数字串个数**

条件：长度n的字符串，只用1\~9，必须包含3和7

| 方案      | 计算        |
| ------- | --------- |
| 总方案     | 9^n（每位9种） |
| 缺3的     | 8^n       |
| 缺7的     | 8^n       |
| 同时缺3和7的 | 7^n       |

**答案 = 9^n - 8^n - 8^n + 7^n**

***

### 3.8 乘法原理（组合计数）

**核心思想：** 如果完成一件事分两步，第一步有m种方法，第二步有n种方法，则总共有 m×n 种方法。

**十六届B5应用：**

- A有 `dp[a]` 种可能（约数个数为a的数有多少个）
- B有 `s[L-a]` 种可能（约数个数 ≤ L-a 的数有多少个）
- **总对数 = Σ dp\[a] × s\[L-a]** ← 用乘法不是加法！

***

### 3.9 置换环分解（Permutation Cycle）

**核心思想：** 任何排列都可以分解成若干个不相交的环（循环）。一个长度为k的环，需要 k-1 次交换才能还原。

**十六届B7：书架还原**

```python
vis = [0] * (n + 1)
ans = 0
for i in range(1, n + 1):
    if not vis[i]:        # 发现新的环
        cur = i
        cycle_len = 0
        while not vis[cur]:
            vis[cur] = 1   # 标记！必须是1不是0
            cur = a[cur]    # 沿着置换关系走
            cycle_len += 1
        ans += cycle_len - 1  # 长度k的环需要k-1次交换
```

例：p = \[3, 1, 2]（位置1→3, 2→1, 3→2）

- 环: 1 → 3 → 2 → 1 （长度3，需2次交换）
- 答案 = 2

***

### 3.10 快速幂（Fast Exponentiation）

**核心思想：** 利用二进制分解，把指数拆成 2^k 的和，通过不断平方，O(log n) 计算 a^b % mod。

**为什么需要快速幂？**

- 普通：a^b 要乘 b 次，b = 10^9 时不可能完成
- 快速幂：只需要 log₂(b) 次，b = 10^9 时只需约 30 次！

**模板：**

```python
def mod_pow(base, exp, mod):
    result = 1
    base = base % mod
    while exp > 0:
        if exp % 2 == 1:           # 当前二进制位是1
            result = (result * base) % mod
        exp = exp >> 1              # 右移一位（除以2）
        base = (base * base) % mod  # 平方
    return result
```

**追踪示例：mod\_pow(2, 13, 1000)**

13的二进制 = 1101 = 8 + 4 + 0 + 1

|   轮次   | exp |  base  |    result   | 操作            |
| :----: | :-: | :----: | :---------: | ------------- |
|    1   | 13奇 |    2   |    1×2=2    | result\*=base |
| <br /> |  →6 |    4   |    <br />   | <br />        |
|    2   |  6偶 |   16   |      2      | 无操作           |
| <br /> |  →3 | <br /> |    <br />   | <br />        |
|    3   |  3偶 |   256  |   2×16=32   | result\*=base |
| <br /> |  →1 | <br /> |    <br />   | <br />        |
|    4   |  1偶 |   256  | 32×256=8192 | result\*=base |
| <br /> |  →0 |   结束   |    <br />   | <br />        |

返回 8192 % 1000 = 192 ✅

### 3.11 逐位贪心 + 位运算（Bitwise Greedy）

**核心思想：** 将答案拆分成二进制的每一位分别计算，每位独立贡献，互不影响。

**十六届B8：异或和最大化** — 求 max{(a\_i XOR a\_j) × (j-i)}

对每一位 k（0\~20）分别处理：

- 维护两组：当前位为0的和当前位为1的
- 新元素加入时，与对面组的所有元素产生贡献

**公式：** 新元素i与对面组的距离之和 = cnt × i - sum

其中 cnt = 对面组元素个数，sum = 对面组元素下标之和

```python
res += (cnt0 * i - sum0) * mask   # mask = 1 << k（第k位的权重）
```

## 五、位运算

### 基本操作符

|  运算符 | 名称   | 示例       |     结果    |
| :--: | ---- | -------- | :-------: |
|  `&` | 按位与  | `5 & 3`  | 1（都为1才为1） |
| `\|` | 按位或  | `5 \| 3` |  7（有1就为1） |
|  `^` | 按位异或 | `5 ^ 3`  |  6（不同为1）  |
|  `~` | 按位取反 | `~5`     |     -6    |
| `<<` | 左移   | `5 << 1` |   10（×2）  |
| `>>` | 右移   | `5 >> 1` |   2（÷2）   |

### 常用技巧

```python
# 取出第k位（0-indexed）
(x >> k) & 1    # 结果为0或1

# 把第k位设为1
x | (1 << k)

# 把第k位设为0
x & ~(1 << k)

# 判断奇偶
x & 1           # 奇数为1，偶数为0

# 2的k次方
1 << k          # = 2^k
```

### 异或性质（十六届B8核心）

```
a ^ a = 0        （自己异或自己=0）
a ^ 0 = a        （异或0不变）
满足交换律和结合律
```

**XOR在B8中的应用：** (a\_i ^ a\_j) 的第k位为1 ⟺ a\_i和a\_j的第k位不同。所以要最大化XOR，希望对应位尽量不同。

## 七、蓝桥杯常用模板

### 模板1：筛法求约数个数

```python
def sieve_divisor_count(L):
    dp = [0] * (L + 1)
    for i in range(1, L + 1):
        for j in range(i, L + 1, i):
            dp[j] += 1
    return dp
```

### 模板2：一维前缀和

```python
def prefix_sum(arr, n):
    s = [0] * (n + 1)
    for i in range(1, n + 1):
        s[i] = s[i - 1] + arr[i]
    return s
```

### 模板3：快速幂

```python
def mod_pow(base, exp, mod):
    result = 1
    base %= mod
    while exp > 0:
        if exp & 1:
            result = result * base % mod
        exp >>= 1
        base = base * base % mod
    return result
```

### 模板4：置换环分解求最小交换次数

```python
def min_swaps(arr, n):
    vis = [0] * (n + 1)
    ans = 0
    for i in range(1, n + 1):
        if not vis[i]:
            cur, length = i, 0
            while not vis[cur]:
                vis[cur] = 1
                cur = arr[cur]
                length += 1
            ans += length - 1
    return ans
```

### 模板5：Counter计频 + 组合数

```python
from collections import Counter
import math

def count_pairs(nums):
    cnt = Counter(nums)
    total = 0
    for v in cnt.values():
        if v >= 2:
            total += math.factorial(v) // math.factorial(v - 2)
    return total
```

### 模板6：逐位处理（通用）

```python
def process_by_bit(arr, n, bit_count=21):
    ans = 0
    for k in range(bit_count):
        mask = 1 << k
        cnt0 = sum0 = cnt1 = sum1 = 0
        for i, x in enumerate(arr):
            if (x >> k) & 1:
                ans += cnt0 * i - sum0
                cnt1 += 1
                sum1 += i
            else:
                ans += cnt1 * i - sum1
                cnt0 += 1
                sum0 += i
        ans *= mask
    return ans
```

## 八、常见错误与避坑指南

### ❌ 错误1：变量名拼写不一致

```python
dic1 = Counter(...)    # 定义的是 dic1
for j in dic:           # ❌ 写成了 dic（少了1）
if djc1[j] > 1:         # ❌ 写成了 djc1（多了c）
```

> **解决：** 定义完变量名后复制粘贴，不要手打

### ❌ 错误2：range起点错误

```python
range(i, L+1, i)       # ✅ 筛法从i开始
range(1, L+1, i)       # ❌ 从1开始会漏掉或重复

range(1, L)             # ✅ 到L-1（因为b至少为1）
range(1, L+1)           # ❌ 多了L导致b=0不合法
```

### ❌ 错误3：漏掉 list() 包裹

```python
num.append(map(int, input().split()))     # ❌ 存的是迭代器
num.append(list(map(int, input().split()))) # ✅ 存的是列表
```

### ❌ 错误4：函数调用忘记括号

```python
delta.total_second    # ❌ 当属性访问了
delta.total_seconds() # ✅ 是方法要加()

Counter[data]          # ❌ []是取下标
Counter(data)          # ✅ ()是调用函数
```

### ❌ 错误5：加法 vs 乘法 vs 幂运算

```python
9 * 10000      # ❌ 90000（普通乘法）
9 ** 10000     # ✅ 9的10000次方（幂运算）
```

### ❌ 错误6：取模遗漏

```python
result = result * base              # ❌ 数字可能爆炸
result = (result * base) % MOD      # ✅ 每步取模
```

### ❌ 错误7：replace不改变原对象

```python
t.replace(second=0)     # ❌ t没变！
t = t.replace(second=0) # ✅ 重新赋值
```

### ❌ 错误8：vis标记值错误

```python
vis[cur] = 0   # ❌ 初始就是0，等于没标记！
vis[cur] = 1   # ✅ 标记为已访问
```

### ❌ 错误9：dp下标错误

```python
dp[i] = dp[i-1] + 1   # ❌ 普通LIS
dp[i] = dp[i-d] + 1   # ✅ 固定间距d的LIS（B6题关键！）
```

### ❌ 错误10：strptime格式不匹配

```python
"%H-%M-%S"            # ❌ 格式期望横杠
"%H:%M:%S"            # ✅ 输入是冒号就用冒号

A + "" + B             # ❌ 空字符串没有空格
A + " " + B            # ✅ 日期和时间之间要有空格
```
